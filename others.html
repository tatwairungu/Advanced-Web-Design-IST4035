Assignment 2 Questions

1. Describe a reason to use embedded styles. Explain where embedded styles are placed on a web page.
Reason to use embedded styles:

Embedded styles are useful when you want to apply CSS rules to a single web page without affecting other pages. This is ideal for one-off designs or custom page-specific styling.

Where embedded styles are placed:

Embedded styles are placed within a <style> tag inside the <head> section of the HTML document.

Example:
<head>
  <style>
    body {
      background-color: lightblue;
    }
  </style>
</head>
2. Describe a reason to use external styles. Explain where external styles are placed and how web pages indicate they are using external styles.
Reason to use external styles:

External styles are ideal for applying consistent styling across multiple web pages. They allow for easier maintenance and better separation of content (HTML) and design (CSS).

Where external styles are placed:

External styles are stored in a separate .css file (e.g., mystyles.css) and are not written directly in the HTML document.

How web pages indicate they are using external styles:

A web page links to an external stylesheet using a <link> tag inside the <head> section of the HTML document.

Example:
<head>
  <link rel="stylesheet" href="mystyles.css">
</head>
